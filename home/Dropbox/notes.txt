KOPS
====
  * export AWS_PROFILE=kops
  * export KOPS_STATE_STORE=s3://clusters.dev.logic-refinery.com
  * kops create cluster --zones=us-east-1d east.dev.logic-refinery.com
  * kops upgrade cluster --name east.dev.logic-refinery.com
  * kops get cluster
  * kops edit cluster east.dev.logic-refinery.com
  * kops edit ig --name=east.dev.logic-refinery.com nodes
  * kops edit ig --name=east.dev.logic-refinery.com master-us-east-1d
  * kops update cluster --name east.dev.logic-refinery.com --yes

  * kops export kubecfg east.dev.logic-refinery.com
  * kops validate cluster
  * kubectl get nodes
  * ssh -i ~/.ssh/id_rsa admin@api.east.dev.logic-refinery.com

  * kops edit ig --name=east.dev.logic-refinery.com nodes
  * kops edit ig --name=east.dev.logic-refinery.com master-us-east-1d
  * kops update cluster east.dev.logic-refinery.com --yes
  * kops rolling-update cluster east.dev.logic-refinery.com

  * kops delete cluster east.dev.logic-refinery.com --yes

    https://kubernetes.io/docs/setup/production-environment/tools/kops/
    https://medium.com/faun/how-to-setup-a-perfect-kubernetes-cluster-using-kops-in-aws-b616bdfae013

    export AWS__PROFILE=kops
    export KOPS_STATE_STORE=s3://clusters.dev.logic-refinery.com
    kops create cluster --zones=us-east-1d useast1.dev.logic-refinery.com

    Suggestions:
     * list clusters with: kops get cluster
     * edit this cluster with: kops edit cluster useast1.dev.logic-refinery.com
     * edit your node instance group: kops edit ig --name=useast1.dev.logic-refinery.com nodes
     * edit your master instance group: kops edit ig --name=useast1.dev.logic-refinery.com master-us-east-1c

    Finally configure your cluster with: kops update cluster --name useast1.dev.logic-refinery.com --yes

    kops export kubecfg useast1.dev.logic-refinery.com

    Suggestions:
     * validate cluster: kops validate cluster
     * list nodes: kubectl get nodes --show-labels
     * ssh to the master: ssh -i ~/.ssh/id_rsa admin@api.useast1.dev.logic-refinery.com
     * the admin user is specific to Debian. If not using Debian please use the appropriate user based on your OS.
     * read about installing addons at: https://github.com/kubernetes/kops/blob/master/docs/addons.md.

    kops edit ig --name=useast1.dev.logic-refinery.com nodes
    kops edit ig --name=useast1.dev.logic-refinery.com master-us-east-1c

    kops update cluster useast1.dev.logic-refinery.com --yes
    kops delete cluster useast1.dev.logic-refinery.com --yes

    nodes max=min=2
    master max=min=1

    kops rolling-update cluster
